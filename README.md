# 🚀 Smart Threat Detection using Machine Learning & Blockchain

## 📌 Problem Statement

With the increasing number of cyber attacks, detecting threats in real-time and ensuring secure logging of those threats has become a major challenge. Traditional systems lack transparency and security in storing threat data.

---

## 🎯 Objective

To develop a system that:

* Detects threats using Machine Learning
* Stores detected threats securely using Blockchain technology
* Ensures data integrity and tamper-proof logging

---

## 🛠️ Tech Stack

* Python (Machine Learning)
* Scikit-learn
* Pandas
* Solidity (Smart Contracts)
* Blockchain (Ethereum-based)

## 🔄 System Workflow

1. Input data is collected from user/system
2. Machine Learning model analyzes the data
3. If threat is detected → flagged as malicious
4. Threat data is stored securely on blockchain

---

## 📊 Machine Learning Model

* Algorithm Used: Random Forest Classifier
* Task: Binary Classification (Normal vs Threat)
* Dataset: Sample dataset (can be extended to real-world datasets)

---

## 🔐 Blockchain Module

* Stores detected threats
* Maintains timestamp of each threat
* Ensures tamper-proof data storage
